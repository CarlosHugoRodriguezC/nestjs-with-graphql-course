# Any List Project

## Dev

1. Clone the repository
2. Copy the `.env.template` and rename as `.env`
3. Run
   ```
   yarn install
   ```
4. Setting up docker image
   ```
   docker compose up -d
   ```
5. Run the nest project with: 
   ```
   yarn start:dev
   ```
6. Visit next url: [local instance of graphql](localhost:3000/graphql)
